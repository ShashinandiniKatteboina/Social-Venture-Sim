📌 Project Description

FounderSim is a full-stack, AI-powered entrepreneurship simulation platform designed to help aspiring founders experience the complete startup journey in a risk-free environment. Instead of learning through theory, users actively build and run a virtual startup by making real-world decisions and observing their consequences in real time.

The platform addresses a critical gap in entrepreneurship education—lack of practical exposure. Many young innovators have great ideas but fail during execution due to limited experience in handling resources, market dynamics, and decision-making under uncertainty.

FounderSim solves this by placing users inside a narrative-driven simulation where they progress through 9 stages of a startup lifecycle, from idea validation to scaling and sustainability. At each stage, users make strategic decisions that directly impact 8 live business metrics, such as user adoption, revenue health, team morale, and investor confidence.

🧠 Key Innovation

The core innovation of FounderSim lies in its AI Character System.

Instead of static feedback, every decision triggers a response from an AI-generated character (e.g., investor, user, co-founder, government officer). These characters:

React based on the user’s entire decision history
Provide context-aware, realistic feedback
Simulate real-world conversations founders face

This makes the experience highly immersive, personalized, and practical.

🎮 How It Works
User selects a social problem and creates a startup idea
Progresses through 9 stages of entrepreneurship
At each stage:
Makes a decision (strategy, product, funding, etc.)
AI character responds with feedback
8 live metrics update instantly
User sees how their choices affect:
Growth
Impact
Sustainability
At the end, an AI-generated Founder Report provides insights and lessons
⚙️ Core Features
📊 Live 8-Metric Dashboard
Tracks real-time startup health (users, revenue, impact, etc.)
🔁 What-If Time Machine
Compare decisions by exploring alternate timelines
⚡ Random Crisis Events
Simulates real startup unpredictability (funding issues, competition, etc.)
🧪 Strategic Sandbox Mode
Test decisions before committing
📄 Founder Debrief Report
Personalized analysis of strengths, weaknesses, and key learnings
🛠️ Tech Stack
Frontend: React + Vite + Tailwind CSS
State Management: Zustand
Backend: Node.js (API layer for AI)
Database: Firestore / PostgreSQL
Authentication: Firebase Auth
AI Engine: Claude API (for dynamic character responses)
Hosting: Vercel + Firebase
